# 月月账单

微信小程序月月账单工具，帮你管理信用卡、花呗、借呗、网贷等债务。

## 功能

- 负债记录：添加信用卡、花呗、借呗、网贷等负债渠道
- 还款管理：记录还款、查看历史还款
- 数据可视化：趋势图、饼图展示债务情况
- 还款日历：查看每月还款日期
- 成就系统：完成还款里程碑获得徽章
- 数据备份：导出/导入JSON格式数据
- 头像昵称：自定义个人资料

## 使用

1. 在微信开发者工具中打开项目
2. 编译预览
3. 添加你的负债记录
4. 记录每次还款

## 技术

- 微信小程序原生开发
- ES5语法
- 本地数据存储
